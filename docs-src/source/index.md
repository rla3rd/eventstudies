# Event Studies Package

Event Studies is an open-source Python package created 
to facilitate the computation of financial event study analyses.

```eval_rst
.. note:: 
  This package is actively maintained and supports Python 3.10 through 3.14.
  
  If you have suggestions to improve and extend this package, please open an issue on `GitHub <https://github.com/rla3rd/eventstudies>`_.
```

## Install

```bash
$ pip install eventstudies
```

Or install from source:

```bash
$ git clone https://github.com/rla3rd/eventstudies.git
$ cd eventstudies/eventstudies
$ pip install -e .
```

## Learn how to use

Go through the [Get started section](get_started.md) to discover through simple 
examples how to use the `eventstudies` package to run your event study for a single event or an aggregate of events.

Read the [API](api/index.md) for more details on functions and their parameters.
