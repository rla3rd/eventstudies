# Introduction

## Introduction

Event Studies package is an open-source python project created 
to facilitate the computation of financial event study analyses.

```eval_rst
.. note::
  This package is stable and actively maintained. The API is well-established and used in production environments.

  If you have suggestions to improve and extend this package, please open an issue on `GitHub <https://github.com/rla3rd/eventstudies>`_.
```
### Install

To install the package run the following command-line in your terminal:

```bash
$ pip install eventstudies
```
